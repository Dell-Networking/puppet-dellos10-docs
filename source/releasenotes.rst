Release 1.0.0
*************

Initial Puppet support for Dell EMC Networking OS10.

- ``os10_image_upgrade``
- ``os10_interface``
- ``os10_lldp``
- ``os10_lldp_interface``
- ``os10_monitor``
- ``os10_route``
- ``os10_snmp``

- None
